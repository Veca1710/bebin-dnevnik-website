- `appimage.png`

- `step-pregled.png`

## Lokalno pokretanje

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Uploaduj ovaj folder na GitHub.
2. U Vercel-u klikni Add New Project.
3. Izaberi GitHub repo.
4. Klikni Deploy.

Vercel će automatski prepoznati Vite projekat.
